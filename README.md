# wellPropertiesPrototype

##First Steps
* install npm
* install bower `npm install -g bower`
* install grunt `npm install -g grunt-cli`
* install less `npm install -g less`
* load npm dependencies `npm install`
* load bower dependencies `bower install`
* run `grunt concat`, `grunt uglify`, `grunt copy`, `grunt less`. Production code is now available inside the `dist` folder.
* for developement run `grunt watch`
